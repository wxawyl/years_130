
# 数字孪生可视化方案与数据模型设计

## 1. 可视化设计方案

### 1.1 设计理念

#### 视觉风格
- **简洁现代**：简洁的线条，现代的审美
- **温暖友好**：使用温暖的色彩，营造亲切的感觉
- **动态响应**：流畅的动画，自然的过渡
- **可定制**：支持多种风格选择

#### 核心视觉元素

| 元素 | 说明 |
|------|------|
| 面部 | 表情丰富，眼神传情 |
| 身体 | 体态变化，肌肉线条 |
| 皮肤 | 光泽变化，反映健康 |
| 特效 | 光晕、粒子、光环 |
| 背景 | 随状态变化的渐变背景 |

### 1.2 状态视觉表现详细设计

#### 1.2.1 睡眠状态视觉映射

**状态1：充足睡眠（健康状态
- **面部：
  - 眼神：明亮有神，瞳孔清晰
  - 肤色：红润有光泽
  - 表情：微笑放松
  - 无黑眼圈、无眼袋

- **身体：
  - 体态：挺拔舒展
  - 肌肉：线条柔和但有精神
  - 能量：充满活力

- **特效：
  - 温暖的金色光晕
  - 缓慢的呼吸动画

状态2：睡眠不足
- **面部：
  - 眼神：疲惫暗淡
  - 肤色：暗淡无光
  - 表情：略显疲惫
  - 轻微黑眼圈

- **身体：
  - 体态：略显慵懒
  - 肌肉：松弛无力
  - 能量：中等偏低

- **特效：
  - 柔和的蓝色调
  - 缓慢的眨眼频率降低

状态3：熬夜
- **面部：
  - 眼神：呆滞无神
  - 肤色：苍白暗淡
  - 表情：萎靡不振
  - 明显黑眼圈和眼袋

- **身体：
  - 体态：耷拉着肩膀
  - 肌肉：完全松弛

- **特效：
  - 暗淡的灰蓝色调

状态4：连续熬夜
- **面部：
  - 眼神：空洞无神

#### 基础动画
| 动画 | 描述 | 触发条件 |
|------|------|---------|
| 呼吸 | 胸部轻微起伏 | 始终播放 |
| 眨眼 | 自然眨眼 | 始终播放 |
| 眼神微动 | 眼神轻微移动 | 始终播放 |

#### 状态切换动画
- **持续：
  - 平滑过渡 300-500ms
  - 使用缓动函数（ease-in-out）

#### 特效动画
- 光晕呼吸
- 粒子漂浮
- 闪烁星星
- 汗珠滑落

#### 2.1.2 TwinVisualParams（视觉参数）
```dart
/// 数字孪生视觉参数
class TwinVisualParams {
  /// 面部状态
  final TwinFaceState face;

  /// 身体状态
  final TwinBodyState body;

  /// 皮肤状态
  final TwinSkinState skin;

  /// 表情
  final TwinExpression expression;

  /// 特效列表
  final List&lt;TwinEffect&gt; effects;

  /// 整体能量值 0.0-1.0
  final double overallEnergy;

  /// 整体健康度 0.0-1.0
  final double overallHealth;

  TwinVisualParams({
    required this.face,
    required this.body,
    required this.skin,
    required this.expression,
    required this.effects,
    required this.overallEnergy,
    required this.overallHealth,
  });

  /// 默认初始状态
  factory TwinVisualParams.initial() {
    return TwinVisualParams(
      face: TwinFaceState.initial(),
      body: TwinBodyState.initial(),
      skin: TwinSkinState.initial(),
      expression: TwinExpression.neutral,
      effects: [],
      overallEnergy: 0.5,
      overallHealth: 0.5,
    );
  }

  /// 线性插值两个状态，用于动画过渡
  TwinVisualParams lerp(TwinVisualParams other, double t) {
    return TwinVisualParams(
      face: face.lerp(other.face, t),
      body: body.lerp(other.body, t),
      skin: skin.lerp(other.skin, t),
      expression: t &lt; 0.5 ? expression : other.expression,
      effects: t &lt; 0.5 ? effects : other.effects,
      overallEnergy: overallEnergy + (other.overallEnergy - overallEnergy) * t,
      overallHealth: overallHealth + (other.overallHealth - overallHealth) * t,
    );
  }

  Map&lt;String, dynamic&gt; toMap() {
    return {
      'face': face.toMap(),
      'body': body.toMap(),
      'skin': skin.toMap(),
      'expression': expression.index,
      'effects': effects.map((e) =&gt; e.toMap()).toList(),
      'overallEnergy': overallEnergy,
      'overallHealth': overallHealth,
    };
  }

  factory TwinVisualParams.fromMap(Map&lt;String, dynamic&gt; map) {
    return TwinVisualParams(
      face: TwinFaceState.fromMap(map['face']),
      body: TwinBodyState.fromMap(map['body']),
      skin: TwinSkinState.fromMap(map['skin']),
      expression: TwinExpression.values[map['expression']],
      effects: (map['effects'] as List).map((e) =&gt; TwinEffect.fromMap(e)).toList(),
      overallEnergy: map['overallEnergy'],
      overallHealth: map['overallHealth'],
    );
  }
}
```

```dart
/// 健康数据快照，用于记录生成数字孪生状态
class HealthDataSnapshot {
  /// 日期
  final DateTime date;

  /// 睡眠数据
  final SleepSnapshot? sleep;

  /// 运动数据
  final ExerciseSnapshot? exercise;

  /// 饮食数据
  final DietSnapshot? diet;

  final MoodSnapshot? mood;

    this.diet,
```
