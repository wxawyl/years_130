## 🆓 食物识别免费本地模型方案

### 方案1：MobileNetV2 + Food-101（推荐）

**数据集**：
- **Food-101**：101种食物，101,000张图片
- **大小**：约5GB
- **许可证**：MIT/Apache 2.0（免费商用

**模型架构**：
- **MobileNetV2（轻量级CNN）
- **准确率**：73-75% Top-1准确率
- **模型大小**：约15-20MB（量化后）

**技术实现**：

```
iOS（Core ML）
- 模型格式：.mlmodel
- 推理速度：~3-5ms（iPhone 15）
- 框架：Vision + Core ML

Android（TensorFlow Lite）
- 模型格式：.tflite
- 推理速度：~4-8ms（Pixel 8 Pro）
- 框架：TensorFlow Lite

### 方案2：MobileNetV2 + YOLOv8n（高精度方案

**架构**：
- YOLOv8n：目标检测（识别食物位置
- MobileNetV2：分类识别
- **准确率**：96.1% precision，92.9% recall

**优点**：
- 支持多食物识别
- 可同时检测多个食物
- 端到端本地化

### 方案3：自定义数据集（推荐）

**建议方案**：
- 基础：Food-101 + 自定义中国菜数据集
- 推荐数据集**：https://www.kaggle.com/datasets/kmader/food41
- 额外补充**：添加常见中餐数据

**训练策略**：
- 迁移学习：基于预训练MobileNetV2
- 数据增强：旋转、缩放、亮度调整
- 量化：INT8量化，保持98%准确率
